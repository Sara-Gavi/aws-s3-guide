# AWS GAME - Static Website Guide

Guía interactiva para entender cómo funciona el **Alojamiento de sitios web estáticos en Amazon S3**.

Interactive guide to understanding how **Static Website Hosting on Amazon S3** works.

## Propósito / Purpose

- **ES:** Practicar el despliegue de contenido estático (HTML/CSS/JS) en la nube de AWS.
- **EN:** Practice deploying static content (HTML/CSS/JS) on the AWS Cloud.

## Tecnologías / Technologies

- **AWS S3**: Hosting del sitio.
- **Vanilla JavaScript**: Lógica del juego.
- **CSS3**: Diseño retro estilo Game Boy.
- **HTML5**: Estructura del sitio.

## Notas finales

**AWS GAME** no pretende ser documentación oficial de AWS.  
Es una herramienta creativa para aprender, explicar y recordar conceptos técnicos de forma más humana.
